Quick start

Verify image
sha256sum -c FedoraX-40-x86_64.iso.sha256
Create USB (example)
sudo dd if=FedoraX-40-x86_64.iso of=/dev/sdX bs=4M status=progress && sync

Install via network

Install packages after first boot with dnf:

sudo dnf update -y
sudo dnf groupinstall "Development Tools" -y
sudo dnf install git podman neovim -y

Installation notes

  1. Create bootable USB (dd, Fedora Media Writer, balenaEtcher).
  2. Boot from USB, choose "Start FedoraX Live" to test without installing.
  3. Run the installer (Anaconda). Pick disk layout, timezone and user.
  4. After first boot, enable 3rd-party repos if needed and update DNF.

Enable RPM Fusion (example)

sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
sudo dnf update -y

Replace $(rpm -E %fedora) with your Fedora release number (e.g. 40).
